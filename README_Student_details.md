# Student Registration Form

A simple **HTML, CSS, and JavaScript** project to create a student registration form with a **CAPTCHA verification**.  

---

## Project Overview
This project allows users to fill in their student details including:
- Name
- Age
- Phone number
- Department
- Gender
- Subjects
- Agreement checkbox  

It also includes a **CAPTCHA** to ensure that the form is submitted by a human.  

---

## Features
- Responsive **form fields** with validation  
- **Department dropdown** for class selection  
- **Gender radio buttons**  
- **Subjects checkboxes**  
- **CAPTCHA** generation and validation  
- Buttons to **refresh CAPTCHA** and **submit**  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## How to Use
1. Clone or download the repository.  
2. Open `index.html` in any browser.  
3. Fill out the student details form.  
4. Enter the CAPTCHA correctly.  
5. Click **Submit** to see the validation alerts.  
6. Use **Refresh CAPTCHA** to generate a new CAPTCHA.
